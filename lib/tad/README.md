# lib/tad — Terrific Audio Driver (vendored, ca65 API only)

Two files from https://github.com/undisbeliever/terrific-audio-driver
(`audio-driver/ca65-api/`, © Marcus Rowe, **Zlib license** — headers intact),
vendored at upstream commit `822164b`.

The SPC700 loader + audio driver binaries are NOT here — they are embedded
inside `assets/audio/tad_audio_data.bin` (generated by `tad-compiler
ca65-export`, which bundles loader + driver + samples + songs in one blob;
`tad_audio_data.asm` exports `Tad_Loader_Bin` / `Tad_AudioDriver_Bin` into
it). To change songs/SFX you need the `tad-compiler` from the same upstream
project; the kit ships a compiled example set (24 songs, 19 SFX — see
`assets/audio/tad_audio_enums.inc`).

API/loader compatibility is proven by the audio run-gate (the loader
handshake + a WAV-recorded non-silent playback), not assumed.
